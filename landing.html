Everything a SIEM pipeline needs

Built on FastAPI with production-grade security, async processing, and AI threat intelligence — ready to receive logs from any stack.

Real-time Ingestion

POST /v1/logs accepts JSON batches and returns a complete anomaly + AI analysis envelope in a single round-trip. P99 under 50ms for typical batches.

Async Bulk Pipeline

POST /v1/logs/bulk accepts gzip-compressed batches up to 10 MB and returns 202 Accepted immediately. Poll /v1/jobs/{id} for results.

4-Category Anomaly Engine

Rules-based scanner inspects every string field across each log line. Detects SQL injection, XSS, brute force, and auth anomalies in milliseconds.

AI Security Analyst

GPT-4o-mini generates threat summaries for CRITICAL/HIGH anomalies. Falls back to a deterministic mock engine when no OpenAI key is set — same JSON schema, always.

Enterprise Security

RFC 7235 authentication, per-key rate limiting (60 req/min), structured audit logging on every request, and body-size caps. Error responses never echo raw API keys.

Stripe Metered Billing

Every ingested line fires a Stripe Billing Meter event automatically. Usage-based pricing — you only pay for what you actually process, with per-tenant Customer ID isolation.

Client SDKs

Zero-dependency libraries for Python and JavaScript — covers sync, async bulk, job polling, and typed error handling out of the box.

Synchronous ingestion
from synthegria import SynthegriaClient

client = SynthegriaClient(
    api_key="synthegria_acme_abc123",
    base_url="https://api.synthegria.io",
)

result = client.ingest([{
    "message": "Failed password for root",
    "source_ip": "203.0.113.5",
}])

print(result["anomaly_count"])       # 1
print(result["ai_analysis"]["summary"])
Async bulk + error handling
from synthegria import AuthError, RateLimitError

# Queue 10 000 events, return immediately
job = client.ingest_bulk(large_batch)
print(job.job_id)

try:
    result = job.wait(timeout=60)
    print(result["compression_ratio"])  # "4.3x"
except AuthError:
    print("Check your API key")
except RateLimitError as e:
    print(f"Retry after {e.retry_after}s")

# Compress and send
echo '[{"message":"event 1"},{"message":"event 2"}]' \
  | gzip \
  | curl -X POST https://api.synthegria.io/v1/logs/bulk \
      -H "X-API-Key: synthegria_acme_abc123" \
      -H "Content-Encoding: gzip" \
      -H "Content-Type: application/json" \
